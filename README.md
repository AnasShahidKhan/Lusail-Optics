### About The Project

This project is a static website built from the ground up to establish a digital presence for Lusail Optics. The goal was to create a clean, modern, and fully responsive site that provides key information to customers, including services, location, and contact details.

This project was built as part of my personal portfolio to demonstrate front-end development, project management, and UI customization skills.


---

### âœ¨ Key Features

* **Fully Responsive Design:** Adapts perfectly to all screen sizes, from mobile phones to desktops.
* **Custom Branding:** The site was fully customized to match the Lusail Optics brand, including a custom logo icon and color palette.
* **Dynamic Content:** Features a 5-card rotating image carousel built with JavaScript to showcase services.
* **Functional Contact Form:** Integrated with **Formspree** to provide a working, secure contact form that sends submissions directly to an email address.
* **Iconography:** Uses **Font Awesome** icons for a clean, professional look in the services and footer sections.
* **Custom Loader:** The default loader was replaced with a custom-animated brand logo for a more professional user experience.
